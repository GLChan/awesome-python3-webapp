# awesome-python3-webapp
python 实战学习(感谢http://www.liaoxuefeng.com/)

开发该Web App需要的第三方库

异步框架aiohttp：

```
$pip3 install aiohttp
```

前端模板引擎jinja2：

```
$ pip3 install jinja2
```

MySQL的Python异步驱动程序aiomysql：

```
$ pip3 install aiomysql
```

### 目录结构
```
awesome-python3-webapp/  <-- 根目录
|
+- backup/               <-- 备份目录

|  |
```
